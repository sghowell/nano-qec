# NanoQEC Human Docs

This section is the human-facing entrypoint for NanoQEC. It explains what the
project is, what the repository implements as of April 9, 2026, how to run it
locally, how to interpret the headline results, and how to collaborate without
needing to read the lower-level harness contracts first.

## Implemented Today

As of April 9, 2026, NanoQEC supports `local-d3-v1` and `local-d5-v1`,
single-host local runs with `uv`, and operator-managed single-host cloud GPU
runs that preserve the documented public CLI and artifact contracts.

## Future Direction

Long-range architecture, autonomy, and research expansion live in
